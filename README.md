
# Remix - Hiking Supply Store 

Store front web app built with React including a fully functional 
shopping cart.

## Practiced Skills

<ul>
    <li>React</li>
    <li>SCSS</li>
    <li>JavaScript</li>
    <li>Linux</li>
    <li>Vim</li>

    <li>Gimp</li>
</ul>

## Getting Started

Clone the repository and install dependencies.

```bash
git clone https://github.com/zachmsdev/Remix-Store.git
cd Shopping-Cart
npm install
npm run dev
```
